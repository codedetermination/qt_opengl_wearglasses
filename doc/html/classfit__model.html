Detailed Description

从给定的图像和3D形状模型中估计3D形状和表情系数。

该应用程序演示了从图像到3d模型的相机估计和拟合。首先，从.pts 文件加载68个关键点，并使用LandmarkMapper将其转换为顶点索引。然后，估计一个正交相机矩阵，然后使用这个相机矩阵，将形状拟合到三维模型。

Parameters
[in]image图像。
