Detailed Description

OpenGL窗口自定义窗口类

OpenGL窗口自定义窗口类，继承自QOpenGLWidget,完成了OpenGL窗口的初始化，绘制，更新等操作，实现了鼠标键盘事件的响应，实现了定时器的功能，并且实现了对于2dtexture和3d模型的加载，绘制，更新等操作
